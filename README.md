# DravenCool Gaming Portal

A comprehensive gaming portal with 100+ unblocked games from various sources (NO 1kh0 games). All games open directly in new tabs for the best gaming experience.

## Features

- 🔓 **No password required** - Direct access to all games
- 🎮 100+ games from various sources (GitHub, Archive.org, Yandex, etc.)
- 🔍 Search functionality
- 📱 Mobile responsive design
- 🚀 Games open directly in new tabs (no iframe loading)
- 🎯 Optimized for Linwize bypass

## How to Use

1. Open `index.html` in your browser
2. Browse games by category or search for specific games
3. Click any game to open it directly in a new tab

## Game Sources

- **GitHub**: Community games from various developers
- **Archive.org**: Classic games via Wayback Machine
- **Yandex**: Popular mobile and web games
- **Emulators**: Console games from various web sources
- **Casual/Puzzle**: Various unblocked gaming sites

## Running Locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Note

Games open in new tabs for better performance and to avoid iframe restrictions that some networks impose.
